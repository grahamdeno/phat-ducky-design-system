# Phat Ducky — Brand & Design System

Brand and visual identity reference for **Phat Ducky**: color palette, typography, logo usage rules, and UI components.

**[View the design system →](index.html)**

## What's in here

```
index.html          the design system itself (open directly, or serve via GitHub Pages)
assets/
  phat-ducky-logo.png   master mascot lockup (web-optimized, 1600px)
tokens/
  colors.css         color tokens as CSS custom properties
  colors.json         color tokens as structured JSON
```

## Viewing it

Open `index.html` in a browser, or enable **GitHub Pages** for this repo (Settings → Pages → Deploy from branch → `main` / root) to get a hosted link.

## Status

- Core palette (Bone, Dusk, Slate, Umber, Ink) is client-supplied and final.
- Accent palette (Gold, Leaf, Grape) is eyedropped from the master artwork — confirm exact hex values once isolated source layers (Illustrator/Photoshop, not the flattened PNG) are available.
- Logo usage documents three lockups: Full Lockup (real asset, embedded), Wordmark Only and Badge/Icon (still placeholders pending separated source files).
